# EEG Signal Analysis Web App

Web-приложение для анализа многоканальных ЭЭГ-сигналов с применением методов машинного обучения.

Проект реализует полный цикл работы с ЭЭГ-данными: загрузку CSV-файлов, визуализацию сигналов, расчёт спектральных характеристик, извлечение признаков, кластеризацию состояний и обучение модели для оценки Valence/Arousal.

## Основные возможности

- загрузка и просмотр многоканальных ЭЭГ-сигналов;
- расчёт PSD и спектральных признаков;
- вычисление относительной мощности delta/theta/alpha/beta-ритмов;
- анализ beta/alpha и theta/alpha отношений;
- кластеризация состояний методом K-means;
- 2D-интерпретация состояния: active/sleep и comfort/discomfort;
- ручная разметка Valence/Arousal;
- обучение ML-модели для предсказания эмоционального состояния;
- хранение результатов в SQLite;
- REST API на FastAPI;
- Web UI для работы с данными.
